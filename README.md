# career-readiness-score-analyzer
A Python and MySQL based web application that evaluates student job readiness using skills, projects, certifications, and CGPA.

## Overview

Career Readiness Score Analyzer is a web application developed using Python, HTML, CSS, and MySQL. The system evaluates a student's job readiness based on skills, projects, certifications, and CGPA.

## Features

- Student profile management
- Readiness score calculation
- Readiness level classification
- Data storage using MySQL
- Simple and user-friendly interface

## Technologies Used

- Python
- HTML
- CSS
- MySQL

## Scoring Criteria

| Parameter | Weight |
|------------|---------|
| Skills | 40 |
| Projects | 25 |
| Certifications | 20 |
| CGPA | 15 |

## Readiness Levels

- Beginner
- Intermediate
- Job Ready
